# CuvilliezAngular

Projet Angular : appli front-end qui affiche la liste des presets récupérée sur le même back-end que le sampler.
Possibilité de modifier le nom du preset, de le supprimer et d'en ajouter un.

## Development server

Lancer l'appli :
```bash
ng serve
```

---
Afin d'avoir la liste des presets il faut lancer le serveur local. Pour cela se placer dans le dossier WebServicesWithCrossDomainSupport et lancer dans le terminal

```
npm run start
```
